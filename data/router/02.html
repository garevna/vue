Объект Route
Текущий маршрут ( активный путь ) Обновляется при каждом навигационном переходе
Доступ к объекту $route
Внутри компонента он доступен как this.$route Изменения маршрута можно отслеживать: 
    watch: {
        '$route': function () {
          ...
        }
    }

           Свойства объекта Route
$route.path тип: string
$route.params тип: Object
Содержит параметры динамических сегментов { id: 'Boom' } При отсутствии параметров пути будет пустым
$route.query тип: Object
Содержит данные строки запроса (query string) Например: 
     { user: "Tornado" }
     для пути /?user="Tornado"
     $route.query.user == "Tornado"
При отсутствии строки запроса будет пустым
$route.hash тип: string
Хэш-часть текущего пути ( после символа # ) Пустая строка, если отсутствует
$route.fullPath тип: string
Полный URL, включая строку запроса и хэш
$route.matched тип: Array
Массив, содержащий объекты маршрутов всех вложенных сегментов текущего пути 
    const router = new VueRouter({
      routes: [
        { path: '/user', component: User,
            children: [
              { path: 'profile', component: Profile }
            ]
        }
      ]
    })
Для URL, равного /user/profile, $route.matched будет массивом, содержащим копии обоих объектов, в порядке сортировки от родителя к потомку.
$route.name
Имя текущего пути (не обязательно, но удобно)
