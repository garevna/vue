Vue Router

Это встроенный компонент Vue В данном примере библиотека vue-router.js подключается тегом: <script src="https://unpkg.com/vue-router/dist/vue-router.js"></script> Получить экземпляр роутера можно с помощью конструктора VueRouter: const router = new VueRouter ( { routes: routes }) При создании экземпляра Vue к нему необходимо привязать созданный ранее роутер: 
new Vue ({
  el: '#app',
  router,
  components: { ... }
})
Экземпляру роутера нужно передать маршруты - объект routes В простом варианте маршрут имеет свойства path, name и component Путь маршрута определяется с помощью свойства path. Установка значения этого свойства в '/' означает маршрут по умолчанию вашего приложения. Для вставки ссылки на маршрут предназначен тег: <router-link to="/маршрут">...текст...</router-link> После рендеринга этот тег будет заменен на обычную гиперссылку Место, куда нужно вставить контент компонента, связанного с маршрутом, должно быть указано с помощью элемента-плейсхолдера: <router-view></router-view> Маршруты могут быть вложенными: 
routes: [
     {
       path: '/...',
       name: '...',
       component: ...
     },
     {
       path: '/firstroute/:name',
       name: 'FirstRoute',
       component: FirstRoute,
       children: [
         {
           path: 'child',
           component: FirstRouteChild
         }
       ]
     }
   ]
