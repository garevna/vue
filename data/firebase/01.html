Подключите Firebase к вашему приложению:

  <script src="https://www.gstatic.com/firebasejs/4.13.0/firebase.js"></script>
  <script>
      // Initialize Firebase
      var config = {
          apiKey: "<...>",
          authDomain: "<...>.firebaseapp.com",
          databaseURL: "<...>",
          projectId: "<...>",
          storageBucket: "<...>",
          messagingSenderId: "<...>"
      }
      firebase.initializeApp ( config )
  </script>

Для получения данных, необходимых для подстановки вместо <...>, нажмите кнопку WEB SETUP или Add Firebase to your web app

Бесплатно:

Authentication (за исключением Phone Auth), Analytics, Predictions, App Indexing, Dynamic Links, Invites, Remote Config, Cloud Messaging (FCM), Performance Monitoring и Crashlytics

Realtime Database: 100 одновременных подключений, 1GB общей "площади"

Cloud Firestore: пропускная способность - 10GB в месяц, 1GB общей "площади, ограничения на чтение/запись/удаление документов в хранилище - 20K/50K/20K в день, соответственно

Storage: 5GB общей "площади, операции: upload - 20K в день, download - 50K в день
