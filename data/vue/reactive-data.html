Различие между данными объектов data и computed заключается в способе изменения их значений

Значения свойств объекта data можно изменять непосредственно, например, с помощью операторов присваивания

Вычисляемые свойства объекта computed являются функциями, возвращающими значение, т.е. их значения вычисляются при вызове функции

Эти функции используют реактивные свойства объекта data для вычисления возвращаемых значений

Используемые в вычислениях свойства объекта data являются зависимостями вычисляемых свойств

Значения свойств объекта computed вычисляются реактивной системой Vue при изменении значений зависимостей

Таким образом, при отсутствии зависимостей свойства объекта computed пересчитываться не будут

        data: {
            ind: 0,
            messages: [
                "Hello",
                "What is your name?",
                "We are glad to meet you"
            ]
        },
        computed: {
            message: () => this.messages [ this.ind ],
            randomVal: () => Math.round ( Math.random () * 100 )
        }

В этом примере свойство message имеет зависимости messages и ind

Каждый раз, когда значение одной из этих зависимостей будет меняться, значение переменной message будет пересчитываться и DOM будет обновляться

Т.е. свойство message будет реактивным

Свойство randomVal не имеет зависимостей, оно не связано ни с одной их переменных объекта data

Значение свойства randomVal пересчитываться не будет, т.е. оно не является реактивным, хотя мы и поместили его в computed
